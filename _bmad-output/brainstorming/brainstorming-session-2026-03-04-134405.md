## 🎯 TỔNG KẾT MỐI QUAN HỆ

### **Quan hệ kế thừa (Inheritance):**
- Registered User kế thừa từ Guest User
- Buyer, Seller, Creator kế thừa từ Registered User

### **Quan hệ bao gồm (Include):**
- Mua NFT **includes** Kết nối ví
- Bán NFT **includes** Kết nối ví
- Tạo NFT **includes** Kết nối ví

### **Quan hệ mở rộng (Extend):**
- Đặt giá chào mua **extends** Xem chi tiết NFT
- Mở gacha **extends** Nhận token hàng ngày

### **Quan hệ tương tác (Association):**
- Buyer ↔ Payment Gateway (mua bằng MoMo/ZaloPay)
- Buyer ↔ Blockchain (xác nhận giao dịch)
- Creator ↔ Smart Contract (mint NFT)
- Admin ↔ Tất cả actors (giám sát)

## PHIÊN BẢN ĐƠN GIẢN HÓA - 4 ACTORS CHÍNH

### DANH SÁCH ACTOR (Đã gộp và bổ sung)

**ACTOR CHÍNH (Primary Actors):**

1. **👤 Người dùng (User)**
   - Gộp: Guest User + Registered User + Buyer + Seller + Creator
   - Mô tả: Tất cả người dùng cuối sử dụng marketplace
   - Vai trò: Xem, mua, bán, tạo NFT, chơi gacha, lai giống

2. **⚙️ Quản trị viên (Admin)**
   - Mô tả: Người quản lý và vận hành nền tảng
   - Vai trò: Quản lý người dùng, kiểm duyệt, cấu hình hệ thống

**ACTOR PHỤ (Secondary Actors - Hệ thống bên ngoài):**

3. **💳 Hệ thống thanh toán (Payment System)**
   - Gộp: MoMo, ZaloPay, Stripe, Banking Gateway
   - Mô tả: Xử lý thanh toán fiat (VND, USD)
   - Vai trò: Nhận tiền, xác nhận thanh toán, chuyển tiền, hoàn tiền

4. **⛓️ Hệ thống Blockchain (Blockchain System)**
   - Gộp: Polygon Network, Smart Contracts, Wallet Providers
   - Mô tả: Xử lý tất cả giao dịch on-chain
   - Vai trò: Mint NFT, transfer ownership, escrow, random generation, breeding
