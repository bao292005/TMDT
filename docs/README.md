# TMDT Docs & Quick Start

README này dành cho người mới: chạy nhanh dự án và biết đọc tài liệu ở đâu.

## Cài đặt và chạy cơ bản
Yêu cầu: **Node.js 18+** và **npm**.

```bash
cd tmdt
npm install
npm run dev
```

Mở trình duyệt tại: **http://localhost:3000**

## Lệnh thường dùng
```bash
npm run lint
npm run build
npm run start
```
